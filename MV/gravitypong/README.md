## Пінг-понг у гравітації

Невеличка класична гра у пін-бол тепер з гравітацією!

Вимоги до гри:

- базова класична реалізація пінг-понгу, однієї із перших комп'ютерних ігор
- різна швидкість гри в залежності від обраної складності
- різна поведінка м'яча в залежності від обраної гравітації

### Розширення гри:

- глобальний редизайн, прибрані залежності від реалізацій, докорінно змінена структура проєкту
- тепер є різні контролери ШІ (від нуба до Про з повним передсказанням поведінки м'яча)
- мінорні зміни для підтримки SOLID-принципів (рендер відірваний від GO, самі GO тепер виконують роль моделей, рендер та поведінка віддаються відповідним системам т. п.)
